Introduction

This example shows that contenteditable attribute can control wether the users can select the nodes. And JavaScript codes can enfore the selection into the user-unselectable nodes. If some JavaScript codes selects into the not-contenteditable nodes, an user click on the selection will reset the selection into a proper one.

DOM Structure

div[contenteditable="true"]
  p[id="content-editable"] contenteditable="true"
  div[id="table-wrapper"]
